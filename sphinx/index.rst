VBA - CONTAINER EDU
===================

Einführung
""""""""""
Diese Schulungsumgebung dient dazu den grundsätzlichen Aufbau und die Bedinung von Docker Container zu verstehen und anwenden.

Im Kapitel Aufbau wird der Aufbau von Docker Containern erklärt und die wichtigsten Begriffe im Zusammenhang mit Docker.

Das Kapitel Basis erklärt die Basis Kommandos mit Docker Container bedient werden können während im Kapitel Fortgeschritten Kommandos angeschaut werden
die mehr ins Detail gehen.

Das Kapitel Dockerfile dient dazu ein Komplettes Docker Image selber zu erstellen.

Während der ganzen Erklärungen macht es Sinn die Kommandos auf der Konsole nachzustellen. Am Ende jedes Kapitel werden mit Übungen die erlernten Kommands wiederhohlt.


.. hint::
    Um auf die Schulungsumgebung zu verbinden muss entweder mit einem SSH Client wie Putty auf die Adresse **127.0.0.1** mit Port **2222** verbinden
    oder im Vagrant Verzeichnis den Command **vagrant ssh** ausführen.

    Der Username und das Passwort ist in beiden Fällen **vagrant**.
